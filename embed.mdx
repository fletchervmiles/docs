# Embed

Seamlessly integrate our services into your application using our embeddable widgets. This page covers everything you need to know about embedding our components.

## Overview

Our embed system allows you to:

- **Seamless Integration** - Drop our widgets directly into your existing UI
- **Customizable Styling** - Match your brand colors and design
- **Responsive Design** - Works on all devices and screen sizes
- **Secure** - All data is encrypted and secure

## Available Embeds

We offer several embeddable components:

### Identity Verification Widget
Embed our complete identity verification flow directly into your application.

### Document Upload Widget
Allow users to upload documents without leaving your application.

### Status Dashboard
Display verification status and progress to your users.

## Basic Implementation

Here's how to embed our basic widget:

```html
<!-- Include our embed script -->
<script src="https://cdn.example.com/embed.js"></script>

<!-- Create a container for the widget -->
<div id="embed-container"></div>

<script>
// Initialize the embed
const embed = new EmbedWidget({
  container: '#embed-container',
  apiKey: 'your_api_key',
  theme: 'light', // or 'dark'
  onComplete: (result) => {
    console.log('Embed completed:', result);
  }
});
</script>
```

## Configuration Options

You can customize the embed with various options:

```javascript
const embed = new EmbedWidget({
  container: '#embed-container',
  apiKey: 'your_api_key',
  
  // Styling options
  theme: 'light', // 'light' or 'dark'
  primaryColor: '#007bff',
  borderRadius: '8px',
  
  // Functional options
  allowedDocuments: ['passport', 'drivers_license'],
  language: 'en',
  autoStart: true,
  
  // Callbacks
  onStart: () => console.log('Started'),
  onComplete: (result) => console.log('Completed:', result),
  onError: (error) => console.log('Error:', error)
});
```

## Styling & Customization

### Theme Options

Choose between light and dark themes:

```javascript
// Light theme
theme: 'light'

// Dark theme
theme: 'dark'
```

### Custom Colors

Match your brand colors:

```javascript
const embed = new EmbedWidget({
  // ... other options
  primaryColor: '#your-brand-color',
  secondaryColor: '#your-secondary-color',
  backgroundColor: '#your-background-color'
});
```

# Franko Embed – Quick Start

## 1 · Copy your embed code

Open the **Connect** tab in the Franko dashboard and choose either:

* **Floating Chat Bubble** (recommended)
* **Custom Trigger** (manual control)

Click **Copy** to grab the snippet generated for your modal.

## 2 · Paste before `</body>`

Place the snippet on every page where you want the chat bubble to appear—ideally just before the closing `</body>` tag.

```html
<!-- Example (Floating Chat Bubble) -->
<script>
  … generated code …
</script>
```

## 3 · (Optional) Add identity verification

To personalise conversations for logged-in users, add a `window.FrankoUser` snippet **before** the embed script, as shown in the [Identity Verification guide](./identity-verification.md).

## 4 · Test your installation

1. Load a page with the embed code.
2. The chat bubble should appear in the bottom-right corner.
3. Click the bubble – the modal should open instantly.

### Troubleshooting

| Symptom | Possible Cause | Fix |
|---------|----------------|-----|
| Bubble not visible | JavaScript error | Open browser console, look for errors. |
| Mixed-content warning | Site served over HTTP | Serve your site and Franko over HTTPS. |
| Modal opens but is blank | CORS or CSP blocks | Ensure `/embed/*` and `/embed.js` are not blocked by your CSP, ad-blockers, or proxy. |
